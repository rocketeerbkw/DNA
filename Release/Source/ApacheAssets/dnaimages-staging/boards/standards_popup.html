Design Standards for Icons

While you can change the default icons we’ve provided in order to fit better with your branding, they do still need to adhere to the standards laid down here:

‘Complain about this message’ icon

The complaints button MUST consist of a triangle with an exclamation mark [!] inside it

‘Unread Item’ icon

The unread item icon used on the Your Page feature MUST contain the word "new"

The unread item icon used on Your Page MUST provide a clear visual indication to the user that there are unread messages. This COULD be achieved with the use of an eight-point star (as is used in the default icon), or by other means.

The unread item icon used on Your Page MUST NOT be a five-point star.
